Walkthrough / Quickstart¶

To get the example up and running without any modifications, you can follow this walkthorugh

  1. Set up the Linux Computer as described in the vendors documentation (for instance : https://www.raspberrypi.org/
  2. Execute the ./Tools/Setup.sh script - This installs all required dependencies
  3. Run the ./Tools/Start.sh script
  4. When the script ends, an URL address is printed. Go to this address using a browser.
  5. You are up and running. Now you can observe the data, make modifications and play around.

Modifications and customization¶

See the ./Tools/Setup.sh script for details. It is well documented and should give you a good explanation on what is going on
