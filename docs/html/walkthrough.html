Walkthrough / Quickstart¶

To get the example up and running without any modifications, you can follow this walkthorugh

  1. Set up the Linux Computer as described in the vendors documentation (for instance : https://www.raspberrypi.org/
  2. Open a terminal and navigate to the RIIM SLIP Example folder
  3. Execute the ./install.sh script - This installs all required dependencies
  4. Run the ./start.sh script
  5. Follow the guide for setting up Node RED described in the Node RED Setup chapter
  6. When the script ends, go to this address using a browser in the raspberry pi : http://localhost:1880/ . If not running the browser from the Raspberry pi, please see the troubleshooting chapter to find the IP address of your edge computer.
  7. You are up and running. Now you can observe the data, make modifications and play around.

Modifications and customization¶

See the ./install.sh script for details. It is well documented and should give you a good explanation on what is going on
